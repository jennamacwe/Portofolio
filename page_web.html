Gizi Checker

Web-Based Nutritional Health Monitoring Application

This web application was developed during an internship at Weza Solutions to fulfill client requirements for a nutritional health monitoring system. This application enables users to calculate BMI, determine daily calorie needs, track daily calorie consumption, monitor their nutritional history over time and also provides an admin panel for managing users and nutritional data.

Tech Stack :

User Dashboard

The dashboard displays a summary of the user’s health data, including BMI history and daily calorie intake.

BMI Checker

This feature allows users to calculate their BMI and see their nutritional status.

Calorie Needs Calculator

Users can calculate their daily calorie needs based on activity and body metrics.

Today’s Calorie Intake

Helps users track daily calorie consumption by meal.

Admin Dashboard

Admins can access all features available in the user interface. The difference is that the admin dashboard displays combined data from all users, allowing admins to monitor BMI inputs, calorie needs, and daily calorie intake across all users.

User Management

Admins can view a list of all registered users who use the Gizi Checker application, making user data management easier.

Nutrition Management

Admins can add nutrition data by importing Excel/CSV files and delete existing nutrition data when needed. This feature ensures that food and calorie data remain up to date.

Food

Stores nutritional data imported by the admin. This table contains food items and their nutritional values, which are used to calculate users’ daily calorie intake.

Users

Stores data of registered users who use the Gizi Checker web application.

Calories

Stores users’ daily calorie intake data based on food consumption recorded for each day.
